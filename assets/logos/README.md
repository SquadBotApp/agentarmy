# AgentArmy Logo Assets

This folder contains logo images for AgentArmy marketing, branding, and other uses.

- `agentarmy_logo_bare.png`: The bare-bones, minimal logo (recommended for most uses)
- `agentarmy_logo_spin1.png` to `agentarmy_logo_spin4.png`: Variations/spins for creative or special use

**Usage:**
- Use the bare logo for official, simple, or default branding.
- Use the spins for creative, marketing, or themed content.
