# AgentArmyOS v1.0

## Overview
AgentArmyOS is a production-grade, aggressive, recursive, multi-agent, multi-platform, self-improving AI OS kernel. It features:
- Six core intelligence subsystems (orchestration, expansion, mobius, reflection, competitive, regulatory)
- Modular provider abstraction (OpenAI, Claude, etc.)
- FastAPI API, Click CLI, and extensible dashboard
- Aggressive recursion, expansion, and strategy/execution alternation
- Robust error handling, compliance, and competitive intelligence

## Key Features
- **Orchestration:** Deterministic, parallel, and recursive agent/task management
- **Expansion:** Tree-based, multi-branch, lesson-driven expansion and escalation
- **Mobius:** Möbius loop for strategy/execution alternation and lens switching
- **Reflection:** Lesson clustering, escalation, and plan restructuring
- **Competitive:** RSS/feeds, trend summarization, actionable proposals
- **Regulatory:** Rule engine, compliance checks, hard stops, safer variants
- **Provider Abstraction:** Dynamic routing, A/B, fallback, multi-provider

## Getting Started
1. **Install dependencies:**
   ```sh
   pip install -r requirements.txt
   ```
2. **Run tests:**
   ```sh
   python -m pytest agentarmy-dashboard/orchestration
   ```
3. **Start the API server:**
   ```sh
   python api/server.py
   ```
4. **Use the CLI:**
   ```sh
   python cli/main.py --help
   ```
